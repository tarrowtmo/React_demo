# TypeScript Frontend Engineer Assignment

## 要求

- 按设计稿实现并保证样式高度还原
- 支持响应式布局：768以上按1440样式展示，768及其以下按768样式展示
- 功能要求：设定一个24小时的倒计时，并实时展示倒计时剩余时间
- 封装为 `<CouponBanner>` 组件
- 使用 Function Component & React Hooks 实现，不能用 Class Component
- 使用 TypeScript 实现
- 设计mock数据并考虑文案支持多语言
